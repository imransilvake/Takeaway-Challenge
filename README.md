# Takeaway Challenge
React seed is based on create-react-app and it incorporates many best practices typically needed in Enterprise apps.


## Content
- [X] SCSS Inclusion

- [X] Build Environments: `dev, stag, next, prod`
- [X] Translation (i18n)
